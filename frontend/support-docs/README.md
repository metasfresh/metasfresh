## Screenshots

The `images/` folder should contain the following screenshots for the documentation:

| Filename | Description | Status |
|----------|-------------|--------|
| `devtools-console.png` | Browser with DevTools open, Console tab selected, showing the hint message | Needed |
| `hilfe-command.png` | Console with `hilfe()` typed in | Needed |
| `download-confirmed.png` | Console showing green success message after download | Needed |

### How to Capture Screenshots

1. Open metasfresh in Chrome/Edge
2. Press F12 to open DevTools
3. Select the Console tab
4. Use Windows Snipping Tool (Win+Shift+S) or browser screenshot feature to capture the relevant areas
5. Save images to the `images/` folder with the names listed above
